# Crawling-Web-Python

## Installation
- clone the repo :
```
$git clone https://github.com/bimbims125/stisla-django.git
```

## Usage
- Create a Virtual Environment :
 ```
 $py -m venv env
 ```
- Activate a Virtual Environment :
 ```
 $env\scripts\activate
 ```
- Install BeautifulSoup4
 ```
 pip install beautifulsoup4
 ```
- Install Pandas
 ```
 pip install pandas
 ```
- Run
 ```
 python crawling.py
 ```
